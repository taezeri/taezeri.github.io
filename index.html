แก้ปัญหาเฉพาะหน้าเกี่ยวกับคอมพิวเตอร์

Computer

การแก้ปัญหาเบื้องต้น

1. การ์ดจอ

อาการ : คอมไม่มีสัญญาณภาพ

สาเหตุ : โดยหลักแล้วมักจะเกิดจากฝุ่นและความชื้น

วิธีแก้ : ต้องมีการถอดออกมาทำความสะอาดหรือใช้น้ำยาไล่ความชื้น

2. พาวเวอร์ซัพพลาย

อาการ : คอมเปิดไม่ติด หรือเปิดติด แต่สักพักคอมดับเอง

สาเหตุ : ไฟไม่เข้า ไม่ทำงานหรือว่าพัดลมไม่หมุน

วิธีแก้ : ควรเปลี่ยนใหม่ เพราะส่วนมาก ไม่นิยมซ่อม ด้วยราคาซ่อมกับซื้อใหม่ ใกล้เคียงกัน

อาการ : เปิดคอมพิวเตอร์ไม่ได้

สาเหตุ : ไม่ได้ต่อคอมพิวเตอร์ลงเต้าเสียบที่ด้านหลัง

วิธีแก้ : ตรวจดูให้แน่ใจว่าสายไฟนั้นเสียบอยู่ที่เต้าเสียบไปบนฝาผนัง และไฟ AC บนฝาผนังที่ลงสายกราวนด์ของคอมพิวเตอร์อย่างแน่นหนา

อาการ : เปิดคอมพิวเตอร์ไม่ได้

สาเหตุ : คอมพิวเตอร์อาจอยูในโหมด Sleep

วิธีแก้ : ตรวจสอบว่าปุ่ม เปิดเครื่องด่วน นั้นสว่างอยู่ และมีไฟสีเหลืองอำพัน ถ้าเป็นเช่นนั้น ให้กดปุ่มดังกล่าวเพื่อออกจากโหมด Sleep

3. ซีพียู

อาการ : คอมเปิดไม่ติด หรือเปิดแล้ว คอมดับเอง

สาเหตุ : ความร้อนของตัวซีพียู

วิธีแก้ : อัดซิลิโคน เพื่อลดความร้อน

4. แรม

อาการ : คอม มีเสียงดัง ติ๊ด

สาเหตุ : ความชื้น ฝุ่น

วิธีแก้ : นำแรมออกมาทำความสะอาด ตรงบริเวณ โลหะ สีทอง ด้านล่าง แล้วลองเสียบเข้าไปใหม่

อาการ : เปิดคอมพิวเตอร์ไม่ได้

สาเหตุ : คอมพิวเตอร์โมดูลหน่วยความจำผิดชนิด

วิธีแก้ : การใช้โมดูลหน่วยความจำอื่นอาจมีผลทำให้การเริ่มระบบยากขึ้นเมื่อคุณอัปเกรดเครื่องคอมพิวเตอร์ คุณจะต้องใช้หน่วยความจำ EDO 60 นาโนวินาที

อาการ : หน่วยความจำไม่เพียงพอที่จะรันโปรแกรม

สาเหตุ : หน่วยความจำไม่ได้รับการตั้งค่าคอนฟิกอย่างเหมาะสมสำหรับโปรแกรม

วิธีแก้ : โปรแกรมจำเป็นต้องใช้หน่วยความจำจำนวนหนึ่งในการรัน ปิดโปรแกรม ใด ๆ ที่กำลังใช้อยู่เพื่อที่จะดูว่าโปรแกรมดังกล่าวจำเป็นต้องใช้หน่วยความจำเท่าใด ให้อ่าน เอกสารที่มาพร้อมกับโปรแกรมนั้น

อาการ : คอมพิวเตอร์แสดงข้อความว่า out of memory

สาเหตุ : ค่าคอนฟิเกอเรชันของหน่วยความจำตั้งค่าที่ไม่ถูกต้อง , หน่วยความจำไม่พอในการรัน

วิธีแก้ : ปิดโปรแกรมที่คุณกำลังใช้อยู่ อ่านเอกสารที่มาพร้อมกับโปรแกรมเพื่อดูข้อกำหนดเกี่ยวกับหน่วยความจำ คุณอาจต้องซื้อและติดตั้งหน่วยความจำเพิ่มเติม

อาการ : คอมพิวเตอร์แสดงข้อความว่า Insufficient memory

สาเหตุ : หน่วยความจำของคอมพิวเตอร์เหลือไม่เพียงพอที่จะรัน โปรแกรม

วิธีแก้ : โปรแกรมบางอย่างจะแอกทีฟอยู่ในแบ็กกราวนด์เมื่อเราเปิดขึ้นมา โปรแกรมเหล่านี้ใช้หน่วยความจำบางส่วนถึงแม้ว่าจะรันอยูในแบ็กกราวด์ก็ตาม

5. ถ่านไบออส

อาการ : เวลาบนเครื่องคอม ไม่ตรง

สาเหตุ : ถ่านหมด

วิธีแก้ : ควรเปลี่ยนใหม่

6. ฮาร์ดดิสก์

อาการ : ข้อความแสดงการผิดพลาดเกี่ยวกับดิสก์ที่ไม่สามารถบูตได้

สาเหตุ : คอมพิวเตอร์พยายามเริ่มระบบจากดิสเก็ตต์ที่ไม่มีซอฟต์แวร์ สำหรับเริ่มระบบ

วิธีแก้ : คอมพิวเตอร์พยายามเริ่มระบบจากดิสเก็ตต์ที่ไม่มีซอฟต์แวร์ สำหรับเริ่มระบบ

อาการ : การทำงานของฮาร์ดไดรฟ์ช้าลง

สาเหตุ : ไฟล์ข้อมูลที่เก็บอยู่บนฮาร์ดดิสก์ อาจอยู่กระจัดกระจาย

วิธีแก้ : ตรวจสอบส่วนของข้อมูลที่หายไปโดยการรันโปรแกรม Disk Defragmenter เพื่อที่จะรันโปรแกรม Disk Defragmenter จากเดสก์ทอปของวินโดวส์ ให้ คลิกที่ปุ่ม Start แล้วชี้ไปที่ Programs จากนั้นชี้ไปที่ Accessories และชี้ไปที่ System Tools ท้ายสุดให้คลิกที่ Disk Defragmenter

อาการ : ไฟแสดงการทำงานของฮาร์ดไดรฟ์สว่างแต่ไม่กระพริบ

สาเหตุ : ไฟล์ข้อมูลที่เก็บอยู่บนฮาร์ดดิสก์ของคุณ อาจจะเสียหาย

วิธีแก้ : ตรวจสอบส่วนของข้อมูลที่หายไปโดยการรันโปรแกรม Disk Defragmenter และเพื่อที่จะรันโปรแกรม Disk Defragmenter6 จากเดสก์ทอปของวินโดวส์ ให้คลิกที่ปุ่ม Start แล้วชี้ไปที่ Programs จากนั้นชี้ไปที่ Accessories และชี้ไปที่ System Tools ท้ายสุดให้คลิกที่ Disk Defragmenter

7. ซีดีรอม

อาการ : คอมพิวเตอร์ไม่สามารถอ่านแผ่นซีดีได้

สาเหตุ : ไม่ได้วางซีดีในไดรฟ์ในตำแหน่งที่ถูกต้อง

วิธีแก้ : ถ้าคุณมีซีดีไดรฟ์แบบโหลดด้วยถาด ให้กดปุ่มนำแผ่นซีดีออก แล้วค่อย ๆ กดแผ่นซีดีลงในตำแหน่ง ที่เหมาะสมจากนั้นโหลดแผ่นเข้าไปใหม่

อาการ : คอมพิวเตอร์ไม่สามารถอ่านแผ่นซีดีได้

สาเหตุ : ระบบไม่รู้จักซีดีไดรฟ์

วิธีแก้ : ปิดคอมพิวเตอร์ แล้วคอยอย่างน้อย 30 วินาที จากนั้นเปิดคอมพิวเตอร์ ขึ้นมาใหม่อีกครั้ง

อาการ : คอมพิวเตอร์ไม่สามารถอ่านแผ่นซีดีได้

สาเหตุ : ใส่แผ่นซีดีกลับข้าง

วิธีแก้ : นำแผ่นซีดีออก แล้วพลิกแผ่นซีดีกลับ จากนั้นโหลดเข้าไปใหม่

อาการ : คอมพิวเตอร์ไม่สามารถอ่านแผ่นซีดีได้

สาเหตุ : แผ่นซีดีสกปรก

วิธีแก้ : ทำความสะอาดแผ่นซีดีด้วยชุดทำความสะอาด

8. ดิสก์เก็ต

อาการ : ไฟแสดงสถานะของดิสเก็ตต์สว่างอยู่

สาเหตุ : มีการใส่แผ่นดิสเก็ตต์อย่างไม่เหมาะสม

วิธีแก้ : นำแผ่นดิสเก็ตต์ออก แล้วใส่เข้าไปใหม่แผ่นดิสเก็ตต์เกิดความเสียหาย ใช้ดิสเก็ตต์แผ่นอื่น หรือตรวจสอบแผ่นดิสเก็ตต์โดยการรันโปรแกรม ScanDisk และเพื่อที่จะรันโปรแกรม ScanDisk จากเดสก์ทอปของวินโดวส์ ให้คลิกที่ปุ่ม Start แล้วชี้ไปที่ Programs จาก นั้นชี้ไปที่ Accessories และชี้ไปที่ System Tools ท้ายสุดให้คลิกที่ ScanDisk ซอฟต์แวร์

อาการ : ดิสเก็ตต์ไดรฟ์ไม่สามารถเขียนลงแผ่นดิสเก็ตต์ได้

สาเหตุ : แผ่นดิสเก็ตต์ยังไม่ได้ฟอร์แมต

วิธีแก้ : ให้ฟอร์แมตแผ่นดิสเก็ตต์

อาการ : ดิสเก็ตต์ไดรฟ์ไม่สามารถเขียนลงแผ่นดิสเก็ตต์ได้

สาเหตุ : ดิสเก็ตต์ไดรฟ์มีการป้องกันการเขียน (Write-Protected)

วิธีแก้ : ยกเลิกการป้องกันการเขียน หรือใช้ดิสเก็ตต์แผ่นอี่นที่ไม่มีการป้องกันการ เขียน และเพื่อที่จะยกเลิกการป้องกันการเขียน ให้เลื่อนแผ่นสีดำที่ด้านหลังของแผ่นดิสเก็ตต์เพื่อปิดรู

อาการ : การเขียนข้อมูลลงไดรฟ์ที่ไม่ถูกต้อง

สาเหตุ : ตรวจสอบตัวอักษรของไดรฟ์ในสเตทเมนท์ของพาธ

วิธีแก้ : พรอมต์ A:\> แสดงว่าเรากำลังทำงานจากไดรฟ์ A ส่วนพรอมต์ C:\> แสดงว่าเรากำลังทำงานจากฮาร์ดไดรฟ์

อาการ : ไม่มีเนื้อที่เพียงพอบนแผ่นดิสเก็ตต์ ดิสเก็ตต์ออก หรือใช้ดิสเก็ตต์แผ่นอื่น

สาเหตุ : ลบข้อมูลบางส่วนที่เก็บอยู่บนแผ่น

วิธีแก้ : ระมัดระวังอย่าลบข้อมูลสำคัญที่เก็บอยู่บนแผ่นดิสเก็ตต์โดยบังเอิญ

อาการ : ดิสก์เก็ตต์ไดรฟ์ไม่สามารถอ่านแผ่นดิสเก็ตต์

สาเหตุ : แผ่นดิสเก็ตต์ ยังไม่ได้ฟอร์แมต

วิธีแก้ : ให้ฟอร์แมตแผ่นฮาร์ดแวร์

อาการ : คอมพิวเตอร์ไม่รู้จักอุปกรณ์ใหม่ว่าเป็นส่วนหนึ่งของคอมพิวเตอร์

สาเหตุ : ไม่ได้ตั้งค่าคอนฟิกคอมพิวเตอร์ของคุณหลังจากติดตั้งอุปกรณ์ใหม่

วิธีแก้ : ถ้าเป็นอุปกรณ์แบบพลักแอนด์เฟลย์ วินโดวส์จะรู้จักอุปกรณ์นั้นและตั้งค่า คอนฟิกให้โดยอัตโนมัติ แต่ถ้าเป็นอุปกรณ์ที่ไม่ใช่พลักแอนด์ใ ห้อ่านเอกสารที่มาพร้อมกับอุปกรณ์สำหรับข้อมูลเกี่ยวกับการตั้งค่าคอนฟิก

อาการ : คอมพิวเตอร์ไม่รู้จักอุปกรณ์ใหม่ว่าเป็นส่วนหนึ่งของคอมพิวเตอร์

สาเหตุ : สายเคเบิลที่เชื่อมต่ออุปกรณ์ใหม่เข้ากับคอมพิวเตอร์นั้นหลวม หรือยังไม่ได้เสียบสายไฟของอุปกรณ์

วิธีแก้ : เสียบสายไป และตรวจดูว่าอุปกรณ์นั้นต่ออยู่กับคอมพิวเตอร์อย่างแน่นหนา

อาการ : เสียบสายไป และตรวจดูว่าอุปกรณ์นั้นต่ออยู่กับคอมพิวเตอร์อย่างแน่นหนา

สาเหตุ : เมื่อคอมพิวเตอร์แนะนำคุณให้เปลี่ยนแปลงค่าคอนฟิเกอเรชันของระบบ

วิธีแก้ : ปิดเครื่องคอมพิวเตอร์ แล้วเปิดใหม่อีกครั้ง คุณจะเห็นข้อคุณไม่ได้ยอมรับการ เปลี่ยนแปลงเหล่านั้น

9. จอภาพ

อาการ : อักษรต่าง ๆ ที่ปรากฏบนหน้าจอมืดมัว

สาเหตุ : ไม่ได้ปรับตัวควบคุมความสว่างอย่างเหมาะสม

วิธีแก้ : ปรับตัวควบคุมความสว่างที่อยู่บนหน้าจอ

อาการ : หน้าจอว่างเปล่า

สาเหตุ : สายเคเบิลที่ต่อเชื่อมจอภาพเข้ากับ คอมพิวเตอร์นั้นหลวมยังหรือไม่ได้เสียบสายจอภาพ

วิธีแก้ : เสียบสายไป และตรวจดูให้แน่ใจว่าช่องเสียบจอภาพนั้นเชื่อมต่ออยู่กับ คอมพิวเตอร์อย่างเหมาะสมและแน่นพอ

อาการ : หน้าจอว่างเปล่า

สาเหตุ : คุณได้ติดตั้งยูทิลิตีที่ทำให้หน้าจอว่างเปล่า

วิธีแก้ : กดคีย์ใด ๆ หรือเคลื่อนไหวเมาส์ หน้าจอปัจจุบันจะปรากฏขึ้นอีกครั้งหนึ่ง

อาการ : หน้าจอว่างเปล่า

สาเหตุ : คอมพิวเตอร์อยู่ในโหมด Sleep

วิธีแก้ : กดปุ่ม เปิดเครื่องด่วน หน้าจอปัจจุบันจะปรากฏขึ้นอีกครั้งหนึ่ง

อาการ : จอภาพร้อนเกินไป

สาเหตุ : พื้นที่สำหรับระบายอากาศไม่เพียงพอให้อากาศถ่ายเทได้สะดวก

วิธีแก้ : เว้นพื้นที่ให้มีข้องระบายอากาศอย่างน้อย 3 นิ้ว (7.6 ซม.) ตรวจดูให้แน่ใจว่าไม่มี อะไรปิดอยู่ด้านบนของจอภาพที่ขัดขวางการหมุนเวียนของ

10. เมาส์

อาการ : เมาส์ไม่ตอบสนองต่อการเคลื่อนไหว

สาเหตุ : สายเมาส์เสียบไม่แน่นหนาในช่องเสียบที่ถูกต้อง ซึ่งอยู่ด้านหลังของคอมพิวเตอร์

วิธีแก้ : ตรวจสอบและเสียบสายเมาส์ให้แน่น

อาการ : เมาส์ไม่ตอบสนองต่อการเคลื่อนไหว

สาเหตุ : ไม่ได้ติดตั้งไดรเวอร์ของเมาส์ หรือ ติดตั้งไดรเวอร์ที่ไม่ถูกต้อง

วิธีแก้ : ตรวจดูให้แน่ใจว่าได้ติดตั้งไดรเวอร์ของเมาส์ที่ถูกต้อง จากเดสก์ทอปของวินโดวส์ให้คลิกที่ปุ่ม Start จากนั้น ชี้ไปที่ Settings แล้วคลิกที่ Control Panel เมื่อเห็นหน้าต่าง Control Panel ให้ดับเบิ้ลคลิกที่ไอคอน Mouse แล้วคลิกที่แท็บ General

11. แฟกซ์/โมเดม

อาการ : แฟกซ์/โมเด็มไม่ตอบสนองต่อซอฟต์แวร์แฟกซ์/โมเด็ม

สาเหตุ : ไม่ได้ต่อแฟกซ์/โมเด็มอยู่อย่างเหมาะสม

วิธีแก้ : ตรวจดูให้แน่ใจว่าได้ต่อแฟกซ์/โมเด็มไว้อย่างแน่นหนา และสายโทรศัพท์ เสียบอยู่ที่เต้าเสียบที่ฝาผนังอย่างแน่นหนา

อาการ : แฟกซ์/โมเด็มไม่ตอบสนองต่อซอฟต์แวร์แฟกซ์/โมเด็ม

สาเหตุ : มีการกำหนดอุปกรณ์มากกว่า 1 ตัวให้กับพอร์ตอนุกรม

วิธีแก้ : ตรวจดูให้แน่ใจว่าไม่มีอุปกรณ์อื่นใช้พอร์ตอนุกรมเดียวกับแฟกซ์/โมเด็ม

อาการ : การสื่อสารด้วยแฟกซ์/โมเด็มเป็นไปอย่างยากลำบาก

สาเหตุ : แฟกซ์/โมเด็มอาจจะเกิความขัดแย้งกับโปรแกรมอื่น

วิธีแก้ : โปรแกรมบางอันอาจจะขัดแย้งกับซอฟต์แวร์แฟกซ์/โมเด็มในตัวของคอมพิวเตอร์ให้ออกจากโปรแกรมใด ๆ ที่คุณกำลังรัน แต่ไม่ได้ใช้งานอยู่

อาการ : การสื่อสารด้วยแฟกซ์/โมเด็มเป็นไปอย่างยากลำบาก

สาเหตุ : เครื่องคอมพิวเตอร์อื่นวางหู

วิธีแก้ : หมุนหมายเลขแฟกซ์ใหม่สายโทรศัพท์มี

อาการ : การสื่อสารด้วยแฟกซ์/โมเด็มเป็นไปอย่างยากลำบาก

สาเหตุ : เสียงดังมาก และการโทรศัพท์ถูกตัด

วิธีแก้ : หมุนหมายเลขแฟกซ์ใหม่ ถ้าปัญหายังคงเกิดอยู่ ให้ตรวจสอบกับ องค์การ โทรศัพท์โดยปกติจะมีฟิลเตอร์ให้มาเพื่อช่วยลดเสียงรบกวนในสาย นอกจากนั้น ให้ตรวจสอบการเชื่อมต่อโทรศัพท์ และแฟกซ์/โมเด็ม ถ้าสายที่ต่ออยู่หลวม แม้ว่าเล็กน้อย ก็อาจจะก่อให้เกิดเสียงรบกวนในสายได้

อาการ : การสื่อสารด้วยแฟกซ์/โมเด็มเป็นไปอย่างยากลำบาก

สาเหตุ : คุณมีสายเรียกซ้อน (Call Waiting) รออยู่ในสายโทรศัพท์ของคุณ สายเรียกซ้อนเข้ามานี้ จะตัดการเชื่อมต่อของโมเด็ม

วิธีแก้ : ในบางพื้นที่ บริการับสายเรียกซ้อนสามารถยกเลิกได้โดยการกด *70 ,ก่อนที่จะหมุน

อาการ : การสื่อสารด้วยแฟกซ์/โมเด็มเป็นไปอย่างยากลำบาก

สาเหตุ : โทรศัพท์พ่วงบนสายโทรศัพท์เดียวกัน

วิธีแก้ : ตรวจดูให้แน่ใจว่าโทรศัพท์ทั้งหมดที่เชื่อมต่ออยู่กับสายโทรศัพท์เดียวกันกับยกหูขึ้นแฟกซ์/โมเด็มวางอยู่ อย่างเหมาะสม

อาการ : ไม่มีเสียงสัญญาณดังขึ้นขณะที่ต่อโมเด็มอยู่

สาเหตุ : คอมพิวเตอร์ใช้โมเด็ม K56flex

วิธีแก้ : ถ้าคุณมีคอมพิวเตอร์ส่วนบุคคลคอมแพครุ่นเพรสซาริโอ 4500 ที่มาพร้อมกับ โมเด็ม K56flex คุณจะไม่ได้ยินสัญญาณใด ๆ ในขณะที่ กำลังเชื่อมต่อโมเด็มข้อมูลเกี่ยวกับ ไวรัสของคอมพิวเตอร์

12. ระบบเสียง

อาการ : คอมพิวเตอร์ไม่มีเสียง

วิธีแก้ : คลิกที่ไอคอน Speaker บนทาสก์บาร์ของวินโดวส์ เมื่อบ็อกซ์ควบคุมระดับเสียง ปรากฏตรวจดูให้แน่ใจว่าเช็คบ็อกซ์ Mute ไม่มีเครื่องหมายใดอยู่มีการลดระดับเสียงให้เบาเพิ่ม ระดับเสียงโดยการกดปุ่มเพิ่มระดับเสียง (+)ที่ด้านหน้าของคอมพิวเตอร์ลำ โพงไม่ได้รับการ เชื่อมต่ออย่าง ตรวจดูให้แน่ในว่าลำโพงเชื่อมต่อกับคอมพิวเตอร์แน่นหนาหรือไม่
